Development Tools

During VAX/VMS and Alpha AXP years we used our own "MicroCODE Development System" (MDS) built as a shell on top of DEC Code Management (CMS) and Module Management System (MMS). With Windows development we use a internal version of our MDS along with Visual Studio. For Web development we use Git and GitHub and Visual Studio Code with our MDS process translated into 'Git Flow'-like workflow.

  • Internal: MDS
  • VAX/VMS: DEC CMS/MMS
  • Windows: VS 2008-2022
